Compare Oxygen-Binding Proteins [Fe bound proteins] via SVM

See:
Oxypred: Prediction and Classification of Oxygen-Binding Proteins 
S. Muthukrishnan, Aarti Garg, and G.P.S. Raghava 
Geno. Prot. Bioinfo. Vol. 5 No. 3–4 2007 

http://www.imtech.res.in/raghava/oxypred/

“Oxypred” article uses (1-4) to compare results: 
1.accuracy, (ACC) = (Σ True positive + Σ True negative) / Σ Total population 
2.specificity, (SPC) true negative rate 
3.sensitivity, true positive rate (TPR) 
4.MCC, Matthew’s correlation coefficient 

See: 
https://en.wikipedia.org/wiki/Sensitivity_and_specificity
